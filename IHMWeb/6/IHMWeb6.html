IHM Web 6: Les méthodes GET et POST

Séance 6 - 14/12/2022

Pendant cette dernière séance, nous avons continué sur les Forms en étudiant les méthodes GET and POST. Ces deux types de requêtes permettent une interaction client/serveur pour transmettre des données entre ordinateurs.

Les deux méthodes se différencie par deux méthodes de transmission différentes. La méthode GET transmet ces données à travers l'URL intégrer après le symbole "?". La méthode plus sécurisée est la méthode POST, elle envoie les données à travers le corps de la requête HTTP et donc permet un transfert de données privées sans peur de dévoilement.
